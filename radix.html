How Radix Sort Works

Radix Sort is a non-comparative sorting algorithm that sorts numbers by processing individual digits. It works by distributing elements into buckets according to their digits, starting from the least significant digit to the most significant.

Unlike comparison-based algorithms, Radix Sort's time complexity is not bounded by O(n log n). It can achieve linear time complexity for fixed-length integers.

Time Complexity: O(d * n) where d is the number of digits and n is the number of elements.

Space Complexity: O(n + k) where k is the range of the input (typically 10 for decimal digits).
