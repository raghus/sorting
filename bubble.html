How Bubble Sort Works

Bubble Sort is a simple sorting algorithm that repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.

The algorithm gets its name because smaller elements "bubble" to the top of the list with each iteration.

Time Complexity: O(n²) in worst and average cases, O(n) in best case (when the array is already sorted).

Space Complexity: O(1) as it only requires a single additional memory space for swapping.

Color Legend
